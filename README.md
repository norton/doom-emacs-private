# doom-emacs-private

My private module for [doom-emacs](https://github.com/hlissner/doom-emacs).

## Install

``` sh
git clone https://github.com/hlissner/doom-emacs-private ~/.config/doom
```

## Notes

- `git init`
- `git remote add doom https://github.com/hlissner/doom-emacs-private`
- `git pull --depth 1 --allow-unrelated-histories --no-rebase doom master`
- `git remote add origin git@github.com:norton/doom-emacs-private.git`
- `git push origin`
